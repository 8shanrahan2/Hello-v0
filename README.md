# Hello v0

A minimal Vercel-ready hello world app using the common v0 stack:

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- shadcn-style UI component structure

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.


## Build

```bash
npm run build
```

## Deploy

Connect this repo to Vercel or run:

```bash
vercel deploy
```

## What this shows

The homepage renders a polished "Hello world from the v0 stack" screen using Tailwind utility classes and a reusable shadcn-style `Button` component.
